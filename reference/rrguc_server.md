# Shiny application server

Defines the server logic of the rrguc Shiny application.

## Usage

``` r
rrguc_server(input, output, session)
```

## Arguments

- input:

  Shiny input object.

- output:

  Shiny output object.

- session:

  Shiny session object.

## Value

No return value. Called for side effects within Shiny.
