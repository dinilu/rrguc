# Format R coefficients table

Internal helper that extracts and rounds R coefficients from RGUC
results.

## Usage

``` r
.rtable_formatting(res)
```

## Arguments

- res:

  A result list returned by
  [`RGUC()`](https://dnietolugilde.com/rrguc/reference/RGUC.md).

## Value

A data frame with group names and R coefficients.
